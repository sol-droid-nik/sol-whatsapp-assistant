# Химикаты SOL / Kiilto — полное и понятное руководство

Ниже — простое объяснение всех основных химикатов, которые используются в SOL.  
Для каждого указано:

1. **Для чего**  
2. **Как использовать (разведение / рекомендации)**  
3. **Осторожно / нельзя**  
4. **Полезные подсказки**

---

# 1) KASPERI  
**Foaming descaler — кислотный пенный декалкер (сильная кислота)**  
**pH: ~1**  
**Разведение: 1–4 мл/л**

## Для чего
- Сильный известковый камень  
- Ржавчина  
- Белые минеральные отложения  
- Жёсткие застарелые налёты в мокрых помещениях

## Как использовать
- Развести по этикетке (обычно 1–4 мл/л)  
- Нанести губкой или пеной  
- Оставить на 1–3 минуты  
- Хорошо смыть водой

## Нельзя / осторожно
- ❌ Не использовать на **мраморе**, натуральном камне, известняке  
- ❌ Не использовать на **алюминии**  
- ❌ Не смешивать с хлоркой  
- Остерегаться брызг — сильная кислота

## Подсказки
- Если Sanitop не справляется — бери Kasperi  
- Отлично снимает старые белые разводы вокруг кранов

---

# 2) SANITOP  
**Descaler for wet rooms — мягкая кислота**  
**pH: ~4**  
**Разведение: 1–4 мл/л**

## Для чего
- Лёгкий и средний известковый налёт  
- Налёт в душевых, раздевалках  
- Ежедневная и регулярная уборка влажных помещений

## Как использовать
- Развести в ведре или распылителе  
- Нанести губкой/тряпкой  
- Через пару минут смыть водой

## Нельзя / осторожно
- На натуральном камне лучше не использовать  
- Сильно застарелый камень удаляет хуже Kasperi

## Подсказки
- Идеально подходит для ежедневного использования в мокрых зонах

---

# 3) BIORINE  
**Biological cleaner for toilets — биологическое средство**  
**pH: 7**  
**Разведение: 4–8 мл/л**

## Для чего
- Туалеты  
- Неприятные запахи  
- Органические загрязнения

## Как использовать
- Развести в ведре или распылителе  
- Нанести, дать подействовать  
- Можно использовать для регулярной уборки туалетов

## Нельзя / осторожно
- Не смешивать с сильными кислотами и хлоркой  
- Ферменты перестают работать, если смешать с другими агрессивными химикатами

## Подсказки
- Отлично устраняет запахи из туалетных зон  
- Для камня в туалетах — сочетать с Sanitop/Kasperi точечно

---

# 4) SARTEK 2  
**Grease and oil remover — мощное щелочное средство**  
**pH: ~13**  
**Разведение: 5–20 мл/л**

## Для чего
- Жир и масло  
- Застарелый кухонный жир  
- Полы с жирными следами  
- Технические масла

## Как использовать
- Развести по необходимости (чем жирнее — тем концентрированнее)  
- Нанести, дать подействовать 1–3 минуты  
- Механически пройтись (моп / щётка)  
- Смыть чистой водой

## Нельзя / осторожно
- ❌ Не использовать на алюминии  
- ❌ Осторожно со свежевыкрашенными поверхностями  
- Может повредить чувствительные материалы при высокой концентрации

## Подсказки
- Лучшее средство для кухни, гриля, вытяжек и масляных следов

---

# 5) SOL TEHOPUHDISTAJA  
**Power Cleaner — средне-сильный щелочной очиститель**  
**pH: ~10.5–12.5**  
**Разведение: ~6 мл/л**

## Для чего
- Сильная грязь на полу  
- Следы обуви  
- Общая глубинная уборка  
- Подходит для машинной уборки (поломоечных)

## Как использовать
- Развести 6 мл/л  
- Нанести, слегка потереть  
- Смыть или пройтись влажным мопом

## Нельзя / осторожно
- Осторожно на чувствительных поверхностях  
- Не использовать на алюминии

## Подсказки
- Если Yleispuhdistaja слабоват — Tehopuhdistaja справится

---

# 6) SOL YLEISPUHDISTAJA  
**Universal cleaner — универсал для ежедневной уборки**  
**pH: ~8.5**  
**Разведение: ~5 мл/л**

## Для чего
- Повседневная уборка полов  
- Столы, поверхности, оборудование  
- Слабая грязь, пыль, следы

## Как использовать
- Развести 5 мл/л  
- Протирать мопом или тряпкой  
- Можно использовать каждый день

## Нельзя / осторожно
- Не справится с сильным жиром или камнем

## Подсказки
- Это главное средство для ежедневной уборки, «по умолчанию»

---

# 7) WINDOW  
**Glass cleaner — средство для стекол**  
**pH: ~8**  
**Разведение: ~10 мл/л**

## Для чего
- Окна  
- Зеркала  
- Хромированная нержавейка  
- Глянцевые поверхности

## Как использовать
- Развести 10 мл/л  
- Нанести распылителем  
- Протереть микрофиброй

## Нельзя / осторожно
- На очень горячих поверхностях могут быть разводы

## Подсказки
- Лучшее средство для блеска без разводов

---

# 8) ASTIANPESU  
**Manual dishwashing — средство для ручного мытья посуды**  
**pH: ~7**  
**Разведение: ~0.8 мл/л**

## Для чего
- Мытьё посуды  
- Инвентаря  
- Корзин, контейнеров

## Как использовать
- Капля в тёплую воду  
- Промыть, смыть, высушить

## Нельзя / осторожно
- Не подходит для полов и поверхностей  
- Не заменяет универсальное средство

## Подсказки
- Отлично работает для удаления простых бытовых загрязнений

---

# Краткая логика выбора средства

## Если КАМЕНЬ / ИЗВЕСТЬ:
- Лёгкий → **Sanitop**  
- Сильный → **Kasperi**

## Если ЖИР:
- Небольшой → **Yleispuhdistaja**  
- Средний/тяжёлый → **Tehopuhdistaja**  
- Кухня / масло → **Sartek 2**

## Если ТУАЛЕТЫ:
- Ежедневно → **Biorine**  
- Камень → точечно **Sanitop** или **Kasperi**

## Если ОКНА:
- **Window**

## Если ПОСУДА:
- **Astianpesu**

---

# Общая безопасность
- Всегда использовать перчатки.  
- Не смешивать кислоту с хлором.  
- Не смешивать сильные кислоты и щёлочи.  
- Всегда соблюдать дозировки.  
- Работать в проветриваемых местах.
